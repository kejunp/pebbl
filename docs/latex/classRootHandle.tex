\doxysubsection{Detailed Description}
RAII handle for managing GC roots. 

\doxylink{classRootHandle}{Root\+Handle} provides automatic registration and deregistration of GC roots. When created, it registers a reference as a root with the GC heap. When destroyed, it automatically removes the root registration.
